# User Service - Frontend

Frontend da aplicação **User Service**, desenvolvido com **React + Vite**, responsável pela interface de gerenciamento de usuários do sistema.

O projeto faz parte de uma aplicação Full Stack composta por:

- Frontend: React + Vite
- Backend: Spring Boot
- Banco de Dados: PostgreSQL

---

# Funcionalidades

- Cadastro de usuários
- Login e autenticação
- Listagem de usuários
- Atualização de dados
- Exclusão de usuários
- Modais de erro e sucesso
- Consumo de API REST
- Integração com backend Spring Boot

---

# Tecnologias Utilizadas

- React
- Vite
- JavaScript
- Bootstrap
- Axios
- React Router DOM
